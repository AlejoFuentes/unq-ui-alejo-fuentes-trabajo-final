Trabajo Final Integrador (TFI) - Alejo Fuentes

## 🚀 Guía de Instalación y Ejecución

### Requisitos Previos
* [Node.js](https://nodejs.org/) instalado en tu computadora.
* Git instalado.

### 1 - Clonar el proyecto
Cloná el repositorio en tu máquina local abriendo la terminal y ejecutando el siguiente comando:

```bash
git clone https://github.com/AlejoFuentes/unq-ui-alejo-fuentes-trabajo-final.git
```
### 2 - Instalar Dependencias
```bash
cd unq-ui-alejo-fuentes-trabajo-final
npm install
```
### 3 - Ejecutar Localmente
```bash
npm run dev
```
